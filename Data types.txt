Converter String para Number
	ParseInt - retorna inteiros
	ParseFloat - retorna números fracionários
	Unario - usando o + na frente, retorna tanto inteiros quanto fracionários
	Number - usando o Number(<qq coisa>)

Converter Number para String
	toString() - usa a variável.toString()
	String(<qq coisa>) - a variável recebe o retorno disso tudo

Converter Number para Boolean
	Boolean(<qq coisa>) - retorna false (quando recebe 0) ou true (qq coisa != 0)
Template Literal
	Usando `` (crase dupla) e ${} (dólar e chaves), nós conseguimos montar o texto sem ficar concatenando ele
